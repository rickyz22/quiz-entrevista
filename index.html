Glosario Técnico Rápido

🔐 Active Directory (AD)

Active Directory (AD):

Directorio centralizado de Microsoft donde se administran usuarios, contraseñas, grupos de seguridad y computadoras de la red corporativa (el "Dominio").

GPO (Group Policy Object):

Políticas de Grupo. Permiten a IT configurar remotamente opciones de Windows (bloquear USB, mapear impresoras, fondos de pantalla) en miles de PCs a la vez desde el servidor.

OU (Organizational Unit):

Unidades Organizativas. Contenedores dentro de AD para agrupar usuarios/PCs por área (ej. OU_Producción, OU_Administración) y aplicarles GPOs específicas.

Unlock Account vs Reset Password:

Unlock: Destildar el candado en la pestaña Account (el usuario recuerda su clave). Reset: Cambiar la clave + marcar "User must change password at next logon".

Login local (.\\usuario):

El prefijo .\\ antes del nombre de usuario le dice a Windows que valide contra la base local (SAM) y no contra el dominio. Útil cuando no hay red.

🌐 Redes y Networking

Dirección IP (IPv4):

Identificador único de un equipo en la red (ej. 192.168.1.50). Puede ser Estática (fija, para servidores/impresoras) o Dinámica (asignada por DHCP).

DNS (Domain Name System):

Traduce nombres (intranet.jayor.local) a IPs. Si la IP funciona pero el nombre no → falla de DNS.

DHCP:

Servicio que asigna automáticamente IPs, máscaras y gateways a los dispositivos de la red.

Gateway (Puerta de Enlace):

El router que permite salir de la red local hacia otras redes o internet.

APIPA (169.254.x.x):

IP de auto-asignación cuando Windows no encuentra servidor DHCP. Significa: "no tengo red".

VLAN (Virtual LAN):

Segmentación lógica de la red en un switch. Aísla el tráfico entre áreas (Producción, Admin, Invitados) mejorando seguridad y rendimiento.

VPN (Virtual Private Network):

Crea un túnel cifrado para acceder a la red corporativa desde fuera. Split Tunnel = solo el tráfico corporativo va por el túnel.

SLA (Service Level Agreement):

Acuerdo de Nivel de Servicio. Define tiempos máximos de respuesta/resolución. Producción y Despacho siempre tienen SLA crítico.

☁️ Office 365

Portal Admin (admin.microsoft.com):

Centro de control de O365. Ahí se gestionan usuarios, licencias (E1/E3/E5), seguridad y configuración de todos los servicios cloud.

OneDrive:

Almacenamiento cloud personal (1TB con E3/E5). Si se llena, deja de sincronizar.

Archivo .OST (Outlook):

Caché local de Outlook. Si se corrompe, el mail deja de sincronizar aunque OWA funcione. Solución: renombrar el .ost o crear perfil nuevo.

MFA / 2FA (Autenticación Multifactor):

Seguridad extra que pide un código (Token) del Authenticator además de la contraseña. Si falla, revisar la vinculación de la app.

MDM (Mobile Device Management):

Administración de dispositivos móviles corporativos. Permite borrado remoto (Remote Wipe) si se pierde un celular.

🪛 Windows Troubleshooting

Event Viewer (Visor de Eventos):

La "caja negra" de Windows. Registra crasheos, errores de apps y del sistema con timestamps y códigos de error detallados.

Modo Seguro (Safe Mode):

Inicia Windows con drivers básicos. Ideal para desinstalar drivers o software defectuoso que causa BSOD.

services.msc:

Consola para ver y gestionar todos los servicios de Windows (Spooler, Windows Update, etc.). Permite iniciar, detener y cambiar el tipo de arranque.

UAC (User Account Control):

Control de cuentas de usuario. Pide permiso de admin para instalar software. Solución: clic derecho → "Ejecutar como administrador".

🔌 Cableado Estructurado

T568B (Norma de cableado):

Estándar de colores para ponchar RJ45. Orden: Blanco-Naranja, Naranja, Blanco-Verde, Azul, Blanco-Azul, Verde, Blanco-Marrón, Marrón.

Cat5e vs Cat6:

Cat5e: 1 Gbps a 100m. Cat6: 1 Gbps con menos crosstalk + 10 Gbps hasta 55m. Para instalaciones nuevas → Cat6 o Cat6a.

Crimpeadora / Cable Tester:

Crimpeadora: Herramienta para fijar hilos al conector RJ45. Tester LAN: Verifica continuidad pin a pin de los 8 hilos.

Switch vs Patch Panel:

Patch Panel: Regleta pasiva de terminación de cables. Switch: Equipo activo inteligente que enruta el tráfico entre los equipos.

⌨️ Comandos CMD Esenciales

ping [ip/dominio]

Verifica conectividad básica con otro equipo. Ej: ping 8.8.8.8

ipconfig /all

Muestra IP, Máscara, Gateway, MAC y DNS del equipo.

ipconfig /flushdns

Limpia caché DNS local. Útil cuando un sitio cambió de IP.

nslookup [dominio]

Consulta al servidor DNS qué IP tiene un nombre. Diagnóstico de resolución de nombres.

tracert [destino]

Muestra todos los saltos (routers) que recorre un paquete hasta llegar al destino.

netstat -an

Lista todas las conexiones de red activas y puertos abiertos. Detecta conexiones sospechosas.

net use Z: \\servidor\carpeta

Mapea una unidad de red. Ideal para scripts de automatización (.bat / GPO).

net user [usuario] [clave]

Gestiona cuentas locales: cambiar contraseña, crear/eliminar usuarios.

whoami

Muestra DOMINIO\usuario de la sesión activa. Indispensable en PCs multiusuario.

gpupdate /force

Fuerza aplicación inmediata de las GPOs de Active Directory sin esperar el ciclo automático.

sfc /scannow

Escanea y repara archivos corruptos del sistema operativo.

chkdsk /f /r

Revisa y repara errores del disco duro. /f = errores lógicos, /r = sectores dañados.

tasklist / taskkill /PID [n] /F

Listar procesos y forzar cierre de uno trabado usando su PID.

🖨️ Hardware e Impresoras

Impresión Térmica (Zebra):

El Ribbon es la cinta con tinta (cara mate). El Cabezal calienta para fijar la tinta. Limpiar con isopropílico previene rayas e impresiones pálidas.

Print Spooler:

Servicio de Windows que gestiona la cola de impresión. Si crashea, borrar archivos en C:\Windows\System32\spool\PRINTERS y reiniciar.

FRP (Factory Reset Protection):

Protección de Google/Android. Si haces Factory Reset sin desvincular la cuenta de Google, el celular queda bloqueado.
