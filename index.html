Glosario Técnico y Comandos

Sistemas y Active Directory

Active Directory (AD):

Directorio centralizado de Microsoft donde se administran los usuarios, contraseñas, grupos de seguridad y computadoras de la red corporativa (el "Dominio").

GPO (Group Policy Object):

Políticas de Grupo. Permiten a IT configurar remotamente opciones de Windows (bloquear USB, mapear impresoras, fondos de pantalla) en miles de PCs a la vez desde el servidor.

Definiciones de Redes

Dirección IP (IPv4):

Es la "matrícula" o identificador único de un equipo en la red (ej. 192.168.1.50). Puede ser Estática (configurada manualmente) o Dinámica (asignada por DHCP).

DNS (Domain Name System):

Actúa como la agenda telefónica de internet. Traduce nombres fáciles de recordar (ej. intranet.jayor.local) a direcciones IP que las computadoras entienden.

DHCP (Dynamic Host Config. Protocol):

Servicio que asigna automáticamente direcciones IP, máscaras y puertas de enlace a los dispositivos que se conectan a la red, evitando conflictos.

Gateway (Puerta de Enlace):

Es el enrutador (router) que permite a los equipos de una red local comunicarse con otras redes externas, como internet o subredes de otras plantas.

APIPA (169.254.x.x):

Si un equipo Windows está configurado por DHCP pero no encuentra ningún servidor que le asigne IP, se autoasigna una dirección en este rango. Significa falla de conexión al DHCP.

Roaming Wi-Fi:

Proceso por el cual una colectora o dispositivo móvil cambia de un Access Point (AP) a otro sin perder conexión. En plantas con racks metálicos y líquidos (jaula de Faraday / atenuación), el roaming puede fallar y desconectar el equipo.

SLA (Service Level Agreement):

Acuerdo de Nivel de Servicio. Establece los tiempos máximos permitidos de respuesta y resolución en IT corporativo. Un bloqueo en producción o despacho tiene el SLA de máxima prioridad.

Comandos de Diagnóstico (CMD)

ping [ip o dominio]

Verifica si hay conectividad básica con otro equipo enviando paquetes de prueba (ICMP). Ej: ping 8.8.8.8.

ipconfig /all

Muestra la configuración de red actual del equipo (IP, Máscara, Gateway, MAC Address y servidores DNS).

ipconfig /flushdns

Limpia la caché DNS local. Útil cuando una web o sistema interno cambió de IP y la PC sigue intentando ir a la dirección vieja.

tracert [ip o dominio]

Muestra la ruta y todos los "saltos" (routers) que da un paquete hasta llegar a su destino.

net use

Comando para conectar, desconectar y configurar conexiones a recursos compartidos. Ej: net use Z: \\servidor\carpeta mapea la unidad Z:. Ideal para scripts de automatización.

nslookup [dominio]

Consulta los servidores DNS para descubrir qué dirección IP está asociada a un nombre de dominio. Fundamental para diagnosticar problemas de resolución de nombres.

gpupdate /force

Fuerza a la PC a descargar y aplicar inmediatamente las últimas políticas de grupo (GPO) desde Active Directory.

tasklist y taskkill /PID [n] /F

tasklist muestra los procesos en ejecución y su número de PID. taskkill fuerza el cierre de un proceso colgado usando su PID.

whoami

Muestra rápidamente el dominio y nombre de usuario que tiene la sesión iniciada actualmente en la consola.

Infraestructura y Hardware Industrial

Switch vs Patch Panel:

El Patch Panel es una regleta pasiva de terminación de cables. El Switch es el equipo activo inteligente que conecta y enruta el tráfico entre los equipos.

Impresión Térmica (Zebra):

Utilizada ampliamente en plantas logísticas. El Ribbon es el rollo de cinta de transferencia que contiene la tinta en una de sus caras (la parte mate). El Cabezal térmico calienta el Ribbon para fijar la tinta en la etiqueta. Limpiar el cabezal regularmente previene rayas blancas e impresiones pálidas.
